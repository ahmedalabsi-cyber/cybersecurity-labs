Cybersecurity Labs & Practice

This repository contains hands-on labs, notes, and basic scripts created as part of my
cybersecurity studies. The focus is on Linux fundamentals, system security, and basic
automation using shell scripting.

Topics include:
- Linux users, groups, and permissions
- File permission management and security
- Basic system hardening concepts
- Simple Bash scripts for system information

All content reflects my own learning and practice.
